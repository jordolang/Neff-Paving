## MOBILE DESIGN SPECIFICATIONS

### Typography (Mobile Optimized)
```
H1: Oswald Bold, 28px (reduced from 36px)
H2: Oswald Medium, 24px (reduced from 28px)
H3: Oswald Medium, 20px (reduced from 24px)
Body: PT Sans Regular, 16px (maintained)
Small: PT Sans Regular, 14px
Caption: PT Sans Regular, 12px

Line Height: 1.5x minimum
Paragraph Spacing: 1.5x line height
```

### Touch Targets
```
Minimum Size: 44x44px
Button Padding: 12px 20px minimum
Form Fields: 44px minimum height
Link Spacing: 8px minimum between
Icon Size: 24px minimum (32px preferred)
```

### Mobile Navigation Patterns
```
Primary: Hamburger menu with slide-out
Secondary: Sticky bottom navigation bar
Breadcrumbs: Horizontal scroll on mobile
Search: Expandable search bar
Filters: Bottom sheet or dropdown
```

### Mobile Performance Optimizations
```
Images: WebP format, lazy loading
Videos: Compressed, poster images
Forms: Progressive disclosure
Maps: Lite mode with expand option
Animations: Reduced motion preferences
Fonts: Subset loading, display swap
```

### Accessibility Features
```
Focus Indicators: 2px yellow outline
Screen Reader: Proper heading hierarchy
Alt Text: Descriptive for all images
Color Contrast: 4.5:1 minimum ratio
Font Size: Respects system preferences
Touch: 44px minimum target size
```

### Mobile-Specific Features
```
Click-to-Call: All phone numbers linked
GPS Integration: Location-based services
Camera Integration: Photo upload capability
Push Notifications: Project updates
Offline Mode: Cached content availability
App-like Experience: PWA implementation
```
