## Mobile Responsive Considerations

### Mobile Hero (320px - 767px)
- Video background remains but with mobile-optimized overlay
- Stacked CTA buttons
- Reduced text size with maintained hierarchy
- Touch-friendly button sizing (44px minimum)

### Mobile Navigation
- Hamburger menu with smooth slide-out animation
- Full-screen overlay with large touch targets
- Emergency contact prominently displayed

### Mobile Features
- Swipeable project gallery
- Touch-optimized estimate calculator
- Click-to-call functionality
- Location-based services integration

### Tablet Adjustments (768px - 1023px)
- 2-column layout for service cards
- Maintained video hero with adjusted text positioning
- Touch-optimized form controls
