## Image Specifications

### Gallery Images
- **Format**: JPG
- **Dimensions**: 800x600px minimum
- **Quality**: High quality, professionally shot
- **Style**: Should show before/after comparisons where applicable
- **Content**: Clear view of completed paving work, equipment in action

### Team Photos
- **Format**: JPG
- **Dimensions**: 400x400px (square format)
- **Quality**: Professional headshots
- **Style**: Business professional, consistent lighting and background
- **Background**: Clean, preferably neutral

### About Images
- **Format**: JPG
- **Dimensions**: 600x400px minimum
- **Quality**: Historical photos can be lower resolution but should be clear
- **Style**: Documentary style showing company history

## Placeholder Handling

Until actual images are available, the website will gracefully handle missing images by:
1. Showing a placeholder with appropriate alt text
2. Maintaining proper layout structure
3. Using CSS to create attractive placeholders with brand colors

## Image Optimization

All images should be:
- Optimized for web (compressed but maintaining quality)
- Served in WebP format where supported
- Include appropriate lazy loading attributes
- Have meaningful alt text for accessibility

## Brand Guidelines for Photography

- Use warm, professional lighting
- Show equipment and work clearly
- Include safety gear when showing workers
- Maintain consistent color temperature
- Show the quality and attention to detail in finished work
- Before/after shots should clearly demonstrate the transformation
